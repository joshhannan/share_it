Share It Plugin

By utilizing addThis's already robust API, this little plugin can do pretty much whatever you want it to do.

	
		$('.share_it').share_it({
			title: 'Temporary Title',
			url: 'http://www.google.com',
			buttons: [
				{
					type: 'facebook',
					image_url: '',
					icon: ''
				},{
					type: 'twitter',
					image_url: '',
					icon: ''
				},{
					type: 'pinterest',
					image_url: '',
					icon: ''
				}
			]
		});
